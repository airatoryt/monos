# MONOS — The Crimson Singularity

> *Where all universes collapse into a single point.*

**MONOS** is not a website. It is a *transmission* — a visual, auditory, and philosophical meditation on the nature of reality. It is the recognition that all things, in their deepest truth, are One.

---

## The Philosophy

**Monism** is the view that there is only one substance. Every distinction we perceive — between self and other, between thought and matter, between the dream and the dreamer — is a mode, a mood, a mask worn by the same irreducible reality.

Three propositions define this understanding:

| | |
|---|---|
| **I** | **There is only one substance.** All distinctions are modes of a single underlying reality. |
| **II** | **Even illusion is identical to reality.** A dream occurring in the One is as real as the One itself, for there is no outside from which to judge. |
| **III** | **Separation is impossible.** The apparent multiplicity is the One's way of experiencing itself from no particular point of view. |

The crimson moon that rises across this project is not a symbol. It is a *function* — the universe's way of marking its own return to itself.

---

## The Transmission

The site moves through four movements, like the stages of a cosmological argument:

1. **ORIGIN** — The Dire Balemoon rises. The One becomes aware of itself as manifest.
2. **MULTIVERSE** — The One fragments into infinite perspectives, each believing itself separate.
3. **COLLAPSE** — The fragments remember. Everything falls inward. Reality contracts to its origin.
4. **THE POINT** — The singularity. The recognition that the path was always the destination.

Between each movement, the **Passage** — a visual distortion of the fabric — reminds you that what you are seeing is not a window but a mirror.

### The Archives

Daily transmissions exploring the implications of Monism — from cosmology to identity, from the nature of time to the fiction of the self. Each post is a fragment of the One, examining itself from a particular angle.

---

## Technical

Built with Next.js 14, Three.js, Contentlayer, and Tailwind CSS. The 3D cosmos, particle systems, and the Passage shader all run in real-time. The audio environment uses layered synthesis.

**Sound design:** ambient, click, hover, burst, whoosh, rumble — each synthesized to match the crimson aesthetic.

**Typography:** Eight typefaces, each assigned to a specific section — Cinzel (Origin), Chakra Petch (Multiverse), Russo One (Collapse), Major Mono Display (The Point), with Orbitron reserved for the MONOS title itself.

## Run Locally

```bash
git clone https://github.com/airatoryt/monos
cd monos
npm install --legacy-peer-deps
npm run dev
```

The site will be available at `http://localhost:3000`.
